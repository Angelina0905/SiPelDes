# SiPelDes - Sistem Pelayanan Desa

Aplikasi web untuk pelayanan administrasi desa secara online.

## Fitur
- Pengajuan surat (domisili, usaha, tidak mampu)
- Tracking status pengajuan
- Pengaduan masyarakat

## Tech Stack
- Backend: Python Flask
- Frontend: HTML/CSS/JS
- Database: PostgreSQL (AWS RDS)
- Storage: AWS S3
- Deploy: AWS EC2 + Docker
- CI/CD: GitHub Actions
